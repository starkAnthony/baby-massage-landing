# Bolalar Massaji Onlayn Akademiyasi

Landing page for **Shahnoza Soliyeva Bolalar Massaji Onlayn Akademiyasi** — a premium sales page for an online children's massage course.

## Tech Stack

- React 19 + TypeScript
- Vite
- Tailwind CSS v4
- Lucide React icons

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Build

```bash
npm run build
npm run preview
```

## Landing Page Sections

1. Hero — headline, stats, CTAs
2. Why baby massage — demand & statistics
3. Problems — pain points
4. About author — Shahnoza Soliyeva profile & timeline
5. Audience — who the course is for
6. Modules — 3 accordion modules
7. Learning format — infographic flow
8. Pricing — START / EXPERT / VIP tiers
9. Comparison table
10. Bonuses
11. Results — testimonials carousel
12. Certificate
13. FAQ — 20 questions
14. Lead form + final CTA

## Features

- Mobile-first responsive design
- Sticky CTA on mobile
- SEO meta tags & semantic headings
- Analytics tracking hooks on all CTAs
- Lead form with success state (CRM API placeholder)

## Customization

- Update content in `src/data/content.ts`
- Replace placeholder images with real photos
- Connect lead form to your CRM API in `src/sections/LeadForm.tsx`
- Update contact info in `src/data/content.ts`
